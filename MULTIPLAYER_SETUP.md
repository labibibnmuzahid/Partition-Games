# LCTR Multiplayer Setup

## Quick Start

1. **Start the server:**
   ```bash
   cd server
   npm start
   ```

2. **Open the game:** Open `lctr_page.html` in your browser

3. **Create a game:** Click "Create Online Game" and share the Game ID

4. **Join a game:** Enter a Game ID and click "Join Game"

## How to Play

- Alice (creator) goes first, Bob (joiner) goes second
- Remove entire top row OR entire leftmost column
- Player who takes the last tile wins
- Status shows "Your turn" or "Opponent's turn"
