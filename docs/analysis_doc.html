Analysis Fields Explained (Corner & LCTR)

Modes

Normal: Last move wins. Analysis shows combinatorial game metrics (Grundy, uptimality, depth, etc.).

Misere: Last move loses. Analysis shows a binary Misere Value (Win/Loss). Metrics like uptimality/depth/incentive are not shown in misere mode.

Position Status

N-Position: Winning position. The current player has a winning strategy.

P-Position: Losing position. The current player will lose with optimal opponent play.

Value

Normal: Shown as Grundy Value (g). g = 0 means losing (P), g > 0 means winning (N). Higher g often indicates stronger positions.

Misere: Shown as Misere Value. 1 = winning (N), 0 = losing (P). This is not a Grundy nimber.

Uptimality

Also called "Remoteness". For winning positions: minimum moves to guaranteed win. For losing positions: maximum moves the opponent can force before winning.

Shown only in Normal mode.

Reachable Moves

Corner: Number of non-empty subsets of selectable last cells, i.e., 2^(k) − 1.

LCTR: Number of legal primitive moves available (top row removal and/or left column removal): 0–2.

Game Depth

Maximum possible number of moves until the game ends from this position, assuming both players try to prolong the game.

Shown only in Normal mode.

Reversible Moves

Number of moves that can be immediately undone by the opponent in their next turn, returning to the current position.

Computed for Corner. Hidden in misere mode.

Move Incentive (Min/Max)

Range of how much the Grundy value changes with possible moves. Shows the minimum and maximum change in advantage from available moves.

Shown only in Corner, Normal mode.

Optimal Moves

Normal (Corner): Moves that lead to G = 0 (P-position) for the opponent. Shown as coordinates (e.g., R1C4).

Misere (Corner): Moves that lead to Misere Value = 0 for the opponent.

LCTR: Shown as ROW or COL moves; in misere, they lead to Misere Value = 0 for the opponent.

G-Number History

Chart showing how the advantage evolves over moves.

Normal: Shows g-values across moves.

Misere: Shows Win/Loss markers instead of g-values.

Interaction & Controls

Toggle Analysis: Use the bottom-right [analysis-mode] button to show/hide the panel.

Undo: Use the Undo button (visible in analysis mode) to step back through game history.

Hover Highlights: Hovering over legal selections (Corner) or edges (LCTR) will highlight moves as winning, blunder, or neutral according to the current mode.

Reports

The unified report generator supports both Corner and LCTR and includes a Mode selector. In misere mode, the report shows Misere Value and hides fields that are not meaningful in misere (e.g., uptimality, depth).
