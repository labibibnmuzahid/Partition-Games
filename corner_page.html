How to Play Corner

On your turn, remove the last cell from each row according to these rules:

  • For rows that are different lengths, remove the last cell from each row.
  • For consecutive rows of the same length, only remove the last cell from the last row in each group.

Example: If you have rows [5, 4, 4, 3], only rows 1, 3, and 4 lose their last cell (row 2 is skipped because it's part of a consecutive group with row 3).

The player who makes the last move (clears the board) wins the game.
