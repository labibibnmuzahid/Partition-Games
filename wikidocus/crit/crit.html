CRIT: Column-Row RIT

CRIT is an enhanced version of RIT (Impartial Row/Column Game) that allows players to make moves on both rows and columns, dramatically expanding the strategic possibilities of Young diagram manipulation games.

Game Setup

The game begins with a Young diagram (partition) represented as a left-justified grid of squares, just like in RIT.

Initial partition: [5, 4, 2, 1] ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■

Move Types

Dual Movement System

Unlike RIT which only allows row moves, CRIT gives players the choice between two types of moves on each turn:

Row Moves (Original RIT Rules)

  • Selection Rule: Choose any row whose length is at least as long as the row directly below it
  • Move Rule: Remove one or more blocks from the right end of the chosen row
  • Constraint: Must leave at least one block behind (except the bottom row, which can be completely removed)

Column Moves (New CRIT Rules)

  • Selection Rule: Choose any column whose height is at least as tall as the column directly to its right
  • Move Rule: Remove one or more blocks from the bottom of the chosen column
  • Constraint: Must leave at least one block behind (except the rightmost column, which can be completely removed)

Strategic Implications

Enhanced Decision Space

  • Move Choice: Every turn requires choosing not just where to move, but which type of move to make
  • Shape Control: Row moves preserve column structure; column moves preserve row structure
  • Endgame Tactics: Different move types create different approach angles to terminal positions
  • Pattern Recognition: Players must evaluate both horizontal and vertical constraints simultaneously

Advanced Strategy Concepts

  • Dual Threats: Create positions where opponent faces unfavorable choices in both directions
  • Movement Blocking: Use one move type to limit opponent's options in the other direction
  • Tempo Control: Switch between row and column moves to maintain initiative
  • Shape Preservation: Maintain favorable diagram properties through strategic move type selection

Game Theory Properties

CRIT maintains the same theoretical foundation as RIT:

  • Impartial Game: Both players have identical move options
  • Perfect Information: All game state information is visible to both players
  • Normal Play: Last player to move wins
  • Finite Game Tree: Every game must eventually terminate
Computational Complexity: The addition of column moves significantly increases the branching factor, making CRIT computationally more complex than RIT while preserving the Sprague-Grundy theoretical framework.

Victory Conditions

A player wins by making the last legal move.

The game ends when no legal moves remain for either row operations or column operations, creating a null position where the previous player (who made the final move) is declared the winner.

Comparison with RIT

Enhanced Features:

  • Move Variety: Approximately doubles the number of available moves per position
  • Strategic Depth: Requires evaluation of both horizontal and vertical game dynamics
  • Pattern Complexity: Creates more diverse intermediate positions and endgame scenarios
  • Skill Expression: Rewards players who can think multi-dimensionally about Young diagram manipulation

Preserved Elements:

  • Core Mechanics: Maintains the elegance and mathematical properties of partition games
  • Learning Curve: RIT knowledge directly transfers to understanding row moves in CRIT
  • Theoretical Foundation: Still amenable to Grundy number analysis and perfect play algorithms
Playing Tip: Master RIT first to understand row dynamics, then gradually incorporate column thinking. The key to CRIT mastery is recognizing when each move type offers the greatest strategic advantage.
