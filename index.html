Web Application on Integer Partition Games

This project is being written under Rhodes College Summer Fellowship Program '25. Under supervision of Professor Gottlieb, Soumtiro and Aayan (see developer's section for more info) are crafting this web application. The project aims to aid Prof. Gottlieb and his team's research on combinatorial game, and make the games approachable to the folklore.
